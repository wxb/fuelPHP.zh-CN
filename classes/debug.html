Debug 類別

Debug 類別是藉由顯示輸出資訊以除錯變數、物件、陣列等的簡易工具集， 這並不意味著你所有的除錯需求是一站式服務，但做為簡易方法快速拋出資訊以確認你在做的事。

dump(*$args)

dump 方法以格式化的方式並包含回溯資訊輸出多個混合值到瀏覽器中，因此你知道它來自何處。

靜態 是
參數 無限的混合變數
回傳 空
範例 
Debug::dump($stuff, 'junk', array('whatever'));

預設情況下，巢狀結構將顯示為折疊以最小化在頁面佈局輸出的影響， 你可以修改此行為並藉由設定 Debug::$js_toggle_open 為 true 來擴充它。

backtrace()

backtrace 方法顯示檔案及方法被呼叫的行數，以及所有前面的清單。

靜態 是
參數 無
回傳 空
範例 
Debug::backtrace();

classes()

靜態 是
參數 無
回傳 空
範例
