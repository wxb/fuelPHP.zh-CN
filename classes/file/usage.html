File 類別

File 類別提供一系列的方法與檔案＆目錄協作。 可以透過一些輔助方法完成，或透過更進階的物件導向方法， 其中所有的檔案和目錄被包裹在物件。

用法

File 類別的基礎用法是透過以下所列它自己的輔助方法。

exists($path, $area = null)

檢查檔案是否存在。如果 $path 指向一個符號連結，該符號連結會被解析來確保它所指向的是否為一個檔案。

靜態 是
參數
參數 預設 描述
$path 必要 要檢查檔案的位置
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
回傳 布林 true 如果 $path 中的檔案存在，如果不存在的話為 false
拋出 OutsideAreaException 當路徑不在檔案區域內。 FileAccessException 當區域已被使用而且有一個擴充侵犯。
範例 
$exists = File::exists(DOCROOT.'/test.txt');

create($basepath, $name, $contents = null, $area = null)

建立一個檔案。

靜態 是
參數
參數 預設 描述
$basepath 必要 要建立新檔案的位置
$name 必要 給新檔案的檔案名稱
$contents null 想要放進新檔案的內容
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
回傳 true
拋出 FileAccessException 當檔案已經存在。 InvalidPathException 當基礎路徑無效或不可寫入。
範例 
File::create(DOCROOT, 'test.txt', 'Contents for file.');

create_dir($basepath, $name, $chmod = 0777, $area = null)

建立一個目錄。

靜態 是
參數
參數 預設 描述
$basepath 必要 要建立新目錄的位置
$name 必要 給新目錄的名稱
$chmod null 給新目錄的權限
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
回傳 布林 - PHP 函式 mkdir() 的輸出
拋出 InvalidPathException 當基礎路徑不存在或不可寫入。 FileAccessException 當基礎路徑不可寫入。
範例 
File::create_dir(DOCROOT, 'test', 0755);

read($path, $as_string = false, $area = null)

讀取一個檔案並回傳它（$as_string == true）或添加到輸出（$as_string == false）。

靜態 是
參數
參數 預設 描述
$path 必要 要讀取檔案的位置
$as_string false 讓它在 false 時使用 readfile()，或在 true 時使用 file_get_contents()
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
回傳 整數｜字串 - 回傳位元組數當 $as_string 設為 false， 或檔案的內容做為一個字串當它設為 true。
拋出 InvalidPathException 當檔案不存在。
範例 
// 回傳 'Contents for file.' 如果你在 File::create() 的範例中建立該檔案
$file_content = File::read(DOCROOT.'test.txt', true);

read_dir($path, $depth = 0, $filter = null, $area = null)

讀取目錄內容進入一個陣列。

靜態 是
參數
參數 預設 描述
$path 必要 要讀取的目錄位置
$depth 0 子目錄要重現多深：0 是無限、 1 是只有目前目錄或指定數字。
$filter null 要讀取檔案的過濾器：array() 全部，或 null 全部但除了隱藏。
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
回傳 陣列 - 一個有子目錄在上的自然字母排序的陣列。 目錄有一個帶陣列值的字串鍵（或 false 當因為 $depth 限制而沒讀取時）以及帶檔案名稱做為值的檔案整數索引。 所有目錄有一個目錄分隔符做為前綴。
拋出 InvalidPathException 當路徑不是目錄。 FileAccessException 當路徑不可讀取。
範例 
// 讀取一個目錄
$contents = File::read_dir(DOCROOT);

// 有限制深度的讀取
$contents = File::read_dir(DOCROOT, 2);

// 有自訂規則的讀取
$contents = File::read_dir(DOCROOT, 0, array(
	'!^\.', // 不要隱藏檔案／目錄
	'!^private' => 'dir', // 不要 private 目錄
	'\.png$' => 'file', // 只取得 png
	'\.css$' => 'file', // 或 CSS 檔案
	'!^_', // 排除任何以底線開頭的。
));

/**
 * 正如你在上面可以看到的，
 * 你可以制定目錄或檔案的規則，
 * 透過在規則後面指定
 * 添加 "=> 'file'" 或 "=> 'dir'"。
 */

更新一個檔案。

靜態 是
參數

rename($path, $new_path, $source_area = null, $target_area = null)

重新命名或移動一個目錄或檔案。

靜態 是
參數
參數 預設 描述
$path 必要 要重新命名的檔案或目錄位置
$new_path 必要 檔案或目錄的新名稱
$source_area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
$target_area null 要使用的選擇性 Area 物件如果 $source_area 不包含 $new_path，預設是主要配置 （更多資訊請見進階用法）
回傳 布林 rename() 的結果
拋出 OutsideAreaException 當路徑不在檔案區域內。 FileAccessException 當檔案類型不允許。
範例 
// 重新命名 'test.txt' 為 'newname.txt'，並移動它到一個子目錄
File::rename(DOCROOT.'test.txt', DOCROOT.'folder/newname.txt');

// 使用單一區域重新命名
File::rename(DOCROOT.'one.jpg', DOCROOT.'final.jpg', 'docroot');

// 在兩個限制的區域之間移動並重新命名
File::rename(APPPATH.'tmp/file.jpg', DOCROOT.'folder/image.jpg', 'tmp', 'public');

copy($path, $new_path, $area = null, $source_area = null, $target_area = null)

複製一個檔案。

靜態 是
參數
參數 預設 描述
$path 必要 要複製的檔案位置
$new_path 必要 要複製到，包含檔案名稱，的完整路徑
$source_area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
$target_area null 要使用的選擇性 Area 物件如果 $source_area 不包含 $new_path，預設是主要配置 （更多資訊請見進階用法）
回傳 布林 copy() 的結果
拋出 OutsideAreaException 當路徑不在檔案區域內。 FileAccessException 當檔案類型不允許。 InvalidPathException 當來源路徑不是一個檔案。 FileAccessException 當目標檔案已存在。
範例

File::copy(DOCROOT.'one.jpg', DOCROOT.'img/one.jpg', 'public');

// 在兩個限制的區域之間複製
File::copy(APPPATH.'tmp/1.jpg', DOCROOT.'folder/1.jpg', 'tmp', 'public');

download($path, $name = null, $mime = null, $area = null, $delete = false, $disposition = 'attachment')

輸出一個檔案、內嵌或做為一個下載附件。。

靜態 是
參數
參數 預設 描述
$path 必要 檔案路徑
$name null 覆寫檔案的預設檔名
$mime null 覆寫檔案的預設 MIME 類型
$area null 要使用的區域物件，預設是主要配置 （更多資訊請見進階用法）
$delete false 在下載之後是否要刪除檔案
$disposition 'attachment' 是否下載或內嵌輸出。接受 'attachment' 或 'inline'，預設為 'attachment'。
回傳 沒有 - 此方法在初始化下載後結束。
拋出 InvalidPathException 當檔案不存在。 InvalidArgumentException 如果該檔案沒有檔案資訊可以被檢索。 FileAccessException 當檔案無法被開啟讀取。
範例 
// 下載檔案
File::download(DOCROOT.'/my_dir/text.txt', 'newfile.txt');

// 輸出一個內嵌圖像，用來做為一個 <img> 標籤來源
File::download(DOCROOT.'/img/image.jpg', null, 'image/jpeg', null, false, 'inline');

請注意，當你呼叫此方法，它不會立即下載檔案。它會在檔案下載後安排一個 shutdown 事件， 並終止該指令碼。這確保了所有 Fuel shutdown 函式， 像更新 session，會正常執行。
