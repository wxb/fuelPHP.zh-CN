Crypt 類別

Crypt 類別允許加密或解密一個字串。Crypt 類別也可用於內部例如 Fuel Session 類別。

它使用由 PHPSecLib 提供的加密和雜湊方法，所以它不是依賴外部模組如 mcrypt 來作用的。

配置

Crypt 類別透過 app/config/crypt.php 配置檔案做配置，當你第一次使用 Crypt 類別，或如果缺少必要配置值之一時，它將會被產生和填充隨機值。

請注意，這將需要寫入 app/config/crypt.php！如果這是辦不到的，確認所有配置設定已經設好！

以下配置設定可以被定義：

參數 類型 預設 描述
crypto_key 字串 沒有 用於加密程序的隨機加密鍵值，確認你設定的值是唯一且隨機！
crypto_iv 字串 沒有 用於加密程序的隨機加密初始向量，確認你設定的值是唯一且隨機！
crypto_hmac 字串 沒有 用於雜湊訊息確認碼（HMAC）程序的隨機值，確認你設定的值是唯一且隨機！

如果你手動指派鍵，注意它們是 base64_encoded，且長度必須是 4 的倍數以讓它能夠被解碼，確認長度是正確的！

encode($value, $key = false)

encode 方法加密一個字串值，選擇性的使用自訂鍵。

靜態 否
參數
參數 預設 描述
$value 必要 要加密的字串值。
$key 
false
 選擇性的自訂鍵值以使用於加密傳遞的值，如果為 false，配置值 'crypto_key' 會被使用。
回傳 字串
範例 
// 使用自訂鍵加密一個變數
$value = Crypt::encode($value, 'R@nd0mK~Y');

decode($value, $key = false)

decode 方法解密一個字串值，選擇性的使用自訂鍵。

靜態 否
參數
參數 預設 描述
$value 必要 要解密的字串值。
$key 
false
 選擇性的自訂鍵值以使用於解密傳遞的值，如果為 false，配置值 'crypto_key' 會被使用。
回傳 混合 - 隨著解密值的字串值，或 false 如果價無法被解密。
範例 
// 使用自訂鍵解密一個變數
$value = Crypt::decode($value, 'R@nd0mK~Y');
