Package 類別

Package 類別能讓你簡單地載入、卸載、檢查套件是否載入、或取得所有已載入的套件。

load($package, $path = null)

load 方法能讓你在執行階段載入單一或更多套件。如果套件無法找到將會拋出一個 PackageNotFoundException。

靜態 是
參數
參數 預設 描述
$package 必要 要載入的套件名稱，或一個套件與套件路徑的陣列。
$path PKGPATH 指向套件安裝所在的文件夾路徑。
回傳 無
範例 
// 載入 orm 套件
Package::load('orm');

// 載入 parser 套件從特定的目錄
Package::load('parser', '/path/to/packages/dir/');

// 載入多個套件從單一的套件安裝
Package::load( array('First' => PKGPATH.'my'.DS.'first'.DS, 'Last' => PKGPATH.'my'.DS.'last'.DS) );

// 載入不存在的套件
Package::load('awesome'); // 拋出一個 PackageNotFoundException

unload($package)

unload 方法能讓你在執行階段卸載套件。

靜態 是
參數
參數 預設 描述
$package 必要 要卸載的套件名稱。
回傳 無
範例 
// 卸載 orm 套件
Package::unload('orm');

loaded($package = null)

loaded 方法能讓你檢查是否套件已經被載入。如果沒有給套件名稱，會回傳所有已載入的套件。

靜態 是
參數
參數 預設 描述
$package null 要檢查的套件名稱。
回傳 布林｜陣列
範例 
// 檢查 orm 套件是否載入
$loaded = Package::loaded('orm');

// 取得所有已載入套件
$loaded = Package::loaded();
/*
回傳結果如：
array(
    'orm' => '/path/to/orm',
    'parser' => '/path/to/parser',
)
*/
