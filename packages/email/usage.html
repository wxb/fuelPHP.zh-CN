使用 Email 套件

Email 套件只需要你三樣東西便可開始發送電子郵件。

  1. 「發送來源」地址。
  2. 發送對象地址。
  3. 以及最後但並非最不重要的，訊息。

讓我們來發送郵件。

// 建立一個實例
$email = Email::forge();

// 設定來源地址
$email->from('my@email.me', 'My Name');

// 設定接收地址
$email->to('receiver@elsewhere.co.uk', 'Johny Squid');

// 設定一個主題
$email->subject('This is the subject');

// 設定多個接收地址

$email->to(array(
	'example@mail.com',
	'another@mail.com' => 'With a Name',
));

// 並且設定主體。
$email->body('This is my message');

在這裡我們已經準備好開始發送郵件。

例外

發送郵件時，你可能會遭遇兩種類型的例外。

  • \EmailValidationFailedException：當一個或多個電子郵件地址未通過驗證時拋出。
  • \EmailSendingFailedException, 當驅動程序發送失敗時拋出

下面的例子顯示了如何處理例外。

try
{
	$email->send();
}
catch(\EmailValidationFailedException $e)
{
	// 驗證失敗
}
catch(\EmailSendingFailedException $e)
{
	// 驅動程序無法發送信件
}

HTML 郵件。

發送 HTML 郵件與發送一般郵件並沒有太大不同：

// 設定一個 html 主體訊息
$email->html_body(\View::forge('email/template', $email_data));

/** 預設情況下也會從 html 產生一個替代的主體，
	並附加任何內嵌檔案（不像 http://... 的路徑）    **/

// 設定一個替代的主體，這是選擇性的。
$email->alt_body('This is my alt body, for non-html viewers.');

附件

Email 套件支援兩種附件：一般（附件）與內嵌。使用一個內嵌附件， 如果你想在郵件裡使用它的話。例如你可能想在你的郵件裡有一個圖形，並且你想要人們也能在離線看見。 有些客戶端並不快取，是的這很愚蠢。但它是跨網發生這樣的東西。 

// 添加一個附件
$email->attach(DOCROOT.'dir/my_img.png');

// 添加一個內嵌附件
// 在這裡添加一個 cid 指向主體
$email->attach(DOCROOT.'dir/my_img.png', true, 'cid:my_conten_id');

你也可以添加字串附件：

$contents = file_get_contents($my_file);
$email->string_attach($contents, $filename);

預設情況下，HTML 圖片會自動引入，但只引入本地檔案。 看看以下的 HTML 以了解它如何運作。 

<!-- 這會被引入 -->
<img src="path/to/my/file.png" />

<!-- 這不會被引入 -->
<img src="http://remote_host/file.jpeg" />

<!-- 這不會被引入 -->
<img src="cid:my_conten_id" />

你所看見最後的標籤（帶有 cid:）也不會被引入。在這種情況下，郵件希望你自己使用內嵌附件附加文件。

優先權

透過使用 priority 方法修改一個郵件的優先權。 

$email->priority(\Email::P_HIGH);

流水線

流水線是一個你可以一次發送多封 email 的機制。 

$email->pipelining($true);

啟用後，你可以使用相同的 $email 物件來建構並發送多封 email，在使用相同的連線到郵件伺服器。 你應該諮詢你的郵件伺服器託管公司來了解這是否是允許的，以及你在單一流水線可以送出 email 的最大數量。 大多數的主機商限制一個特定數量，之後伺服器斷開連線。

目前，只有 SMTP 驅動支援流水線。

驅動

Mailgun 驅動

Mailgun 是一個由 Rackspace 提供，能讓你依需求量發送電子郵件的服務。你會需要在你的 FuelPHP 中用 composer 安裝 Mailgun 函式庫。

添加下一行到你的 composer.json 檔案並執行更新

"mailgun/mailgun-php": "1.6"
$ composer update

一旦你安裝了套件，你必須要為你的 App 設置配置。


/**
 * 覆蓋預設 Email.php 配置
 */
return array(
    'defaults' => array(
        'driver' => 'mailgun',
        'mailgun' => array(
            'key' => 'YOUR KEY',
            'domain' => 'YOUR DOMAIN'
        ),
    ),
);
