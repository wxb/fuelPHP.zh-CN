建立查詢

注： 在撰寫時，查詢建立器类別会被重新改动，所以他們可以处理多个 SQL 語支。

資料庫查詢 SQL 撰寫的抽象化是藉由 查詢建立器 達成， 查詢建立器由一系列类別組成，每一个处理建立查詢過程中特定的部分。

使用查詢建立器将幾乎忽視像 DB 类別 为建立新查詢建立器物件所提供的快捷方式。建議使用这些方法來取得正確的查詢建立器。 这是常見的做法，并会保持你程式码的可讀性。

查詢建立器由以下的类別組成：

  • Query_Builder（抽象）
  • Query_Builder_Where（抽象）
  • Query_Builder_Select
  • Query_Builder_Insert
  • Query_Builder_Update
  • Query_Builder_Delete
  • Query_Builder_Join
