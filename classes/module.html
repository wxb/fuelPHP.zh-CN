Module 類別

Module 類別能讓你簡易的載入、卸載、檢查一個模組是否存在或是否已被載入，或取得一個所有已載入的模組列表。

load($module, $path = null)

load 方法能讓你在執行階段載入一個或更多模組，如果模組無法被找到，一個 ModuleNotFoundException 將被拋出。

靜態 是
參數
參數 預設 描述
$module 必要 要被載入的模組名稱，或一個模組陣列。
$path null 指向所給模組的路徑，你可以使用它從定義的模組路徑外部載入模組。
回傳 空
範例 
// 載入 users 模組
Module::load('users');

// 從指定的目錄載入 users 模組
Module::load('users', '/path/to/modules/users/');

// 從單一模組安裝程序載入多個模組
Module::load( array('Users' => '/path/to/modules/users/', 'Groups' => '/path/to/modules/groups/') );

// 載入不存在的模組
Module::load('awesome'); // 拋出一個 ModuleNotFoundException

unload($module)

unload 方法能讓你在執行階段卸載一個模組。

靜態 是
參數
參數 預設 描述
$module 必要 要被卸載的模組名稱。
回傳 空
範例 
// 卸載 users 模組
Module::unload('users');

這將移除所有已載入指向此模組的路由！

loaded($module = null)

loaded 方法能讓你檢查一個模組目前是否已載入。如果沒給模組名稱，會回傳所有已載入的模組。

靜態 是
參數
參數 預設 描述
$module null 要被檢查的模組名稱。
回傳 布林｜陣列
範例 
// 檢查 users 模組是否已載入
$loaded = Module::loaded('users');

// 取得所有已載入的模組
$loaded = Module::loaded();
/*
回傳一些東西像：
array(
    'users' => '/path/to/users',
    'groups' => '/path/to/groups',
)
*/

exists($module)

exists 方法能讓你檢查一個模組是否存在。換句話說，是否它能在配置模組路徑之中被找到。 如果找到的話，會回傳指向模組的路徑，否則的話回傳 false。

靜態 是
參數
參數 預設 描述
$module null 要被檢查的模組名稱。
回傳 布林｜字串
範例 
// 選擇性的載入 comments 模組
if (Module::exists('comments'))
{
	Module::load('comments');
}
