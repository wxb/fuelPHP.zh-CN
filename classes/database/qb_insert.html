Query_Builder_Insert 類別

（擴充 Query_Builder_Where）

Query_Builder_Insert 類別為查詢建立程序處理所有 insert 操作，它擴充 Query_Builder_Where 類別，所以所有方法都繼承。

table($table)

table 方法設定／改變要新增的資料表。

靜態 否
參數
參數 類型 預設 描述
$table 字串 必要 資料表名稱
回傳 回傳目前實例。
範例 
// 預處理一個 insert 語句
$query = DB::insert('users');

// 更新要新增的資料表
$query->table('admins');

// INSERT INTO `admins` ...

columns(array $columns)

columns 方法設定／改變要新增的行。

靜態 否
參數
參數 類型 預設 描述
$columns 陣列 必要 行名稱的陣列
回傳 回傳目前實例。
範例 
// 預處理一個 insert 語句
$query = DB::insert('users');

// 設定行
$query->columns(array(
	'name',
	'surname',
	'email')
);

// INSERT INTO `users` (`name`, `surname`, `email`)

values(array $values)

values 方法附加要新增的值。

靜態 否
參數
參數 類型 預設 描述
$values 陣列 必要 值的列表
回傳 回傳目前實例。
範例 
// 預處理一個 insert 語句
$query = DB::insert('users');

// 設定行
$query->columns(array(
	'name',
	'surname',
	'email',
));

// 設定值

$query->values(array(
	'John',
	'Doe',
	'john@doe.com',
));

// INSERT INTO `users` (`name`, `surname`, `email`) VALUES ("John", "Doe", "john@doe.com")

set(array $pairs)

set 方法設定行並附加要新增的值。

靜態 否
參數
參數 類型 預設 描述
$pairs 陣列 必要 行和值的關聯陣列
回傳 回傳目前實例。
範例 
// 預處理一個 insert 語句
$query = DB::insert('users');

// 設定行和值
$query->set(array(
	'name' => 'John',
	'surname' => 'Doe',
	'email' => 'john@doe.com',
));

// INSERT INTO `users` (`name`, `surname`, `email`) VALUES ("John", "Doe", "john@doe.com")

compile(\Database_Connection$db)

compile 方法回傳 insert SQL 查詢做為一個字串。

靜態 否
參數
參數 類型 預設 描述
$db 物件 必要 一個資料庫連線
回傳 回傳 SQL 查詢做為一個字串。
範例 
// 預處理一個 insert 語句
$query = DB::select('*')->from('users');

// 設定限制和偏移
$query->limit(10);
$query->offset(5);

// 取得資料庫連線
$connection = Database_Connection::instance();

// 取得 SQL 查詢
$sql = $query->compile($connection);

reset()

reset 方法重置目前實例的所有值。

靜態 否
參數 無
回傳 回傳目前實例。
範例 
// 預處理一個 insert 語句
$query = DB::insert('users');

// 設定一些值
$query->set(array(
	'name' => 'John',
	'surname' => 'Doe',
));

// 重置它
$query->reset();

// 設定新的值
$query->table('admins')->set(array(
	'name' => 'Mark',
	'surname' => 'Handriks'
));

// SQL 查詢現在會是：
// INSERT INTO `admins` (`name`, `surname`) VALUES ("Mark", "Hendriks")
