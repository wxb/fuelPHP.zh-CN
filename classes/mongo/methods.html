Mongo_Db 類別方法

Mongo_Db 類別能讓你與 MongoDB 資料庫互動。雖然在某些情況下， 它與資料庫如 MySQL 的互動類似，也有一些明顯的差異。

所有排序選項（order by 及 indexes）使用 -1、false、'desc' 或 'DESC' 做為 DESC。任何其他值會是 ASC。

instance($name = 'default')

instance 回傳一個新的 Mongo_Db 實例。使用 $name 參數來指定要使用的配置和資料庫。

預設情況下，$name 是 'default'，它需要你有一個在你 db.php 配置檔案中的預設配置。閱讀關於更多 Mongo_Db 配置。

靜態 是
參數
參數 類型 預設 描述
$collection 字串 
'default'
 要使用的配置群組的名稱。
回傳 一個新的 Mongo_Db 實例
拋出 Mongo_DbException，當沒找到所給實例名稱的配置時。
範例 
// 取得預設群組
$mongodb = \Mongo_Db::instance();

// 取得一個自訂群組
$mongodb = \Mongo_Db::instance('my_group');

get 方法從一個集合回傳結果。

靜態 否
參數
參數 類型 預設 描述

 集合名稱。

foreach($result as $user)

	// 用 $user 做一些事

get_cursor($collection = "")

get_cursor 方法回傳指定集合的 mongodb 游標。它提供了比一般 get() 方法更好的地方是，當處理大型資料集做為文件時，一次只會載入它們存取的進入記憶體而不是一次全部。對長時間執行的程序非常有用。

靜態 否
參數
參數 類型 預設 描述
$collection 字串 
''
 集合名稱。
回傳 一個 MongoCursor 物件。
範例 
// 取得一個實例
$mongodb = \Mongo_Db::instance();

$result = $mongodb->get_cursor('documents');
foreach($result as $document)
{
	// 對 $document 做點什麼
}

靜態 否
參數

''
 集合名稱。
$where 陣列

	// 對 $user 做一些事

select($includes = array(), $excludes = array())

select 方法設定在查詢過程中要包含或排除哪些欄位。

為了使用 excludes 陣列，includes 陣列必須為空。

靜態 否
參數
參數 類型 預設 描述
$includes 陣列 
array()
 要包含的欄位陣列
$where 陣列 
array()
 要排除的欄位陣列
回傳 目前 Mongo_Db 實例。
範例 
// 取得一個實例
$mongodb = \Mongo_Db::instance();
// 只選擇這些欄位
$mongodb->select(array(
	'name',
	'surname',
	'email',
));
$users = $mongodb->get('users');

// 或使用排除陣列

$mongodb = \Mongo_Db::instance();
// 只選擇這些欄位
$mongodb->select(array(), array(
	'not_this',
	'or_this',
));
$users = $mongodb->get('users');

  • m = 多行

false

$disable_end_wildcard 布林
