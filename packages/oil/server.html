伺服器

有了 oil 伺服器實现，你甚至不需要安装成熟的網頁伺服器！ 它会使用 PHP 5.4 版開始內建的網頁伺服器。在命令列啟动它，并使用你的瀏覽器去指向你已经指定的主機名稱和連接埠 （預設是 http://localhost:8000）。 

$ php oil server
Listening on http://localhost:8000
Document root is public/
Press Ctrl-C to quit.

當你完成時，按 Ctrl-C 來終止該伺服器。
