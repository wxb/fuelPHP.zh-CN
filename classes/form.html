Form 類別

Form 類別可以被用來建立單獨的表單元素，或建立一個有驗證的完整表單， 後者則是結合了 Fieldset 類別。

建立單獨的表單元素

open($attributes = array(), $hidden = array())

建立一個表單開始標籤。

靜態 是
參數
參數 預設 描述
$attributes 
array()
 一個做為 action 屬性的字串或一個設定的陣列， 如果沒給，action 將使用目前 Uri， 屬性的值將被用作 HTML 標籤特性。
$hidden 
array()
 一個欄位名稱和值的關聯陣列， 將全部被設定為隱藏欄位。
回傳 字串
範例 
// 回傳 <form action="http://mydomain.com/uri/to/form" accept-charset="utf-8" method="post">
echo Form::open('uri/to/form');

// 回傳 <form action="http://google.com/" accept-charset="utf-8" method="get">
echo Form::open(array('action' => 'http://google.com/', 'method' => 'get'));

close()

建立一個表單結束標籤。

靜態 是
參數 （無）
回傳 字串
範例 
// 回傳 </form>
echo Form::close();

input($field, $value = null, $attributes = array())

建立一個 html input 元素，它可以設定使用的欄位名稱、值和標籤的屬性， 或合為一個陣列做為第一個參數。

靜態 是
參數
參數 預設 描述
$field 必要 欄位名稱的字串或標籤屬性的陣列。
$value 
null
 欄位值，當第一個參數是陣列時將被忽略。
$attributes 
array()
 這些將被用作 HTML 標籤特性。
回傳 字串
範例 
echo Form::input('name', 'value', array('style' => 'border: 2px;'));
// 更多可以添加

hidden($field, $value = null, $attributes = array())

這是一個 Form::input() 的別名，它會自動設定 type 屬性為 'hidden'。

csrf()

建立一個 HTML input 元素，帶有自動設為產生的 CSRF 符記的 value 屬性。

password($field, $value = null, $attributes = array())

這是一個 Form::input() 的別名，它會自動設定 type 屬性為 'password'。

radio($field, $value = null, $attributes = array())

這是一個 Form::input() 的別名，它會自動設定 type 屬性為 'radio'。
為了向下相容，此方法也接受：radio($field, $value = null, array $attributes = array())

靜態 是
參數
參數 預設 描述
$field 必要 欄位名稱的字串或標籤屬性的陣列。
$value 
null
 欄位值，當第一個參數是陣列時將被忽略。
$checked 
null
 是否選取（如果是布林）或符合 $value （如果是字串）， 當第一個參數是陣列時將被忽略。
$attributes 
array()
 這些將被用作 HTML 標籤特性。
回傳 字串
範例 
echo Form::label('Male', 'gender');
echo Form::radio('gender', 'Male', true);
echo Form::label('Female', 'gender');
echo Form::radio('gender', 'Female');

select($field, $values = null, $options = array(), $attributes = array())

建立一個 html select 元素，它可以設定使用的欄位名稱、已選擇的值、選項和標籤的屬性， 或合為一個陣列做為第一個參數。

靜態 是
參數
參數 預設 描述
$field 必要 欄位名稱的字串或標籤屬性的陣列。
$values 
null
 選擇的值或多選值的陣列， 當第一個參數是陣列時將被忽略。
$options 
array()
 value=>label 配對的關聯陣列，也可能包含 option 群組如 opt_name=>array()，該陣列包含自己的一組 value=>label 配對。
$attributes 
array()
 這些將被用作 HTML 標籤特性。
回傳 字串
範例 
echo Form::select('country', 'none', array(
	'none' => 'None',
	'europe' => array(
		'uk' => 'United Kingdom',
		'nl' => 'Netherlands'
	),
	'us' => 'United States'
));
// 更多可以添加

fieldset_open($attributes = array(), $legend = null)

建立一個欄位集開始標籤。

靜態 是
參數
參數 預設 描述
$attributes 
array()
 一個設定的陣列；可能使用 'legend' 鍵去設定欄位集的 legend 屬性。 屬性的值將被用作 HTML 標籤特性。
$legend 
字串
 要用作 legend 欄位集選項的字串
回傳 字串
範例 
// 回傳 <fieldset >
echo Form::fieldset_open();

// 回傳 <fieldset class="example-class" id="example-id">
echo Form::fieldset_open(array('class' => 'example-class', 'id' => 'example-id'));

// 回傳 <fieldset class="example-class" id="example-id"><legend>Custom Legend</legend>
echo Form::fieldset_open(array('class' => 'example-class', 'id' => 'example-id'), 'Custom Legend');

// 回傳 <fieldset class="example-class" id="example-id"><legend>Custom Legend</legend>
echo Form::fieldset_open(array('class' => 'example-class', 'id' => 'example-id', 'legend' => 'Custom Legend'));

fieldset_close()

建立一個欄位集結束標籤。

靜態 是
參數 （無）
回傳 字串
範例 
// 回傳 </fieldset>
