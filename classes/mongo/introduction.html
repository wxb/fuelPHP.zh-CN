Mongo_Db 類別

簡介

Mongo_Db 類別提供方法與 Mongo 資料庫 互動。

除了提供通常的新增、選擇、刪除和更新的功能之外， 它有一些管理方法幫助你對資料庫實用操作。

配置

對於每一個 Mongo 資料庫連線在 app/config/db.php 的 Mongo 陣列裡必須有一個配置。配置選項和解釋表列如下， 每個配置須包含主機名稱和資料庫名稱。

參數 類型 必要 描述
hostname 字串 是 主機名稱
database 字串 是 資料庫名稱
port 數字 否 連線使用埠
replicaset 字串 否 用於連線的副本集名稱
username 字串 否 用於認證的使用者名稱，如果沒有已設定的密碼將忽略
password 字串 否 用於認證的密碼，如果沒有已設定的密碼將忽略

配置範例：

// 在 app/config/db.php 裡

		'mongo' => array(
			// 當沒有實例名稱被提供時使用此群組。
			'default' => array(
				'hostname'   => 'localhost',
				'database'   => 'mongo_fuel',
			),

			// 在下面表列你自己的群組。
			'my_mongo_connection' => array(
				'hostname'   => 'localhost',
				'database'   => 'my_db',
				'replicaset' => 'replica',
				'username'   => 'user',
				'password'   => 'p@s$w0rD',
			),
		),

一旦你設定到位，你可以開始使用 MongoDB。

為了使用 MongoDB 這必須在你的伺服器上可用，看看你的伺服器是否有支援 Mongo， 在你的 phpinfo() 尋找 Mongo 段落或詢問你的主機供應商。
