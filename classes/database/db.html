DB 類別

DB 類別能讓你建立及執行資料庫查詢並取得結果。

query($sql, $type = null)

query 方法回傳一個新的 Database_Query_Builder 物件。 確切的物件取決於傳遞的類型。如果沒傳遞類型，Fuel 選擇 DB::SELECT 如果該 SQL 查詢以 'SELECT' 開頭，而且它將回傳一個 Database_Query_Builder_Select 物件。

靜態 是
參數
參數 預設 描述
$query 必要 SQL 查詢
$type null SQL 查詢類型。可能的值有：DB::SELECT、DB::INSERT、DB::UPDATE 和 DB::DELETE。
回傳 回傳一個 Database_Query_Builder 請求類型的物件。
範例 
// 將預處理：SELECT * FROM `users`
$query = DB::query('SELECT * FROM `users`');

請注意，傳遞正確類型是重要的。 資料庫驅動會有不同的對待。 如果你有一個必須回傳一組結果的查詢，但不是 SELECT，使用 DB::SELECT 來確保結果會正確回傳。

last_query()

last_query 方法回傳最後執行的 SQL 查詢。

靜態 是
參數
參數 預設 描述
$db null 資料庫連線。
回傳 最後執行的 SQL 查詢。
範例 
// 執行查詢
$user = DB::select()->from('users')->where('id', 1)->execute();

echo DB::last_query();
// SELECT * FROM `users` WHERE `id` = 1

select()

select 方法回傳一個新的 Database_Query_Builder_Select 物件。

靜態 是
參數
參數 預設 描述
$columns null 要選擇的行。
回傳 回傳一個 Database_Query_Builder_Select 物件。
範例 
// 將預處理：SELECT *
$query = DB::select();

// 將預處理：SELECT `id`, `name`
$query = DB::select('id', 'name');

select_array($columns)

select_array 方法回傳一個新的 Database_Query_Builder_Select 物件。

靜態 是
參數
參數 預設 描述
$columns array(); 要選擇行的陣列。
回傳 回傳一個 Database_Query_Builder_Select 物件。
範例 
// 將預處理：SELECT *

// 將預處理：SELECT `id`, `name`

insert($table = null, $columns = array())

insert 方法回傳一個新的 Database_Query_Builder_Insert 物件。

靜態 是
參數
參數 預設 描述
$table null 要新增資料的資料表。
$columns array(); 要選擇行的陣列。
回傳 回傳一個 Database_Query_Builder_Insert 物件。
範例

$query = DB::insert('table_name');

// 將預處理：INSERT INTO `table_name` (`id`, `name`)

靜態 是

expr($expression)

expr 方法回傳一個新的 Database_Expression 物件。

靜態 是
參數
參數 預設 描述
$expression null 表達式。
回傳 回傳一個 Database_Expression 物件。
範例 
// 回傳 new Database_Expression('COUNT(table_name.table_column)');
$expr = DB::expr('COUNT(table_name.table_column)');

// 在產生查詢的表達式中防止不正確的引號
$expr = DB::expr('columnname + 1');

你必須在你查詢中所有的表達式使用 DB:expr()，換句話說，其中的一切不是一個欄位名稱， 以確保你傳遞的值沒有被包在引號或反引號中。

quote($string, $db = null)

quote 方法回傳一個為 SQL 查詢帶引號的字串。

靜態 是
參數
參數 預設 描述
$string 必要 要上引號的字串
$db null 資料庫連線。
回傳 回傳一個為 SQL 查詢帶引號的字串。
範例 
// 回傳 'something\'s quoted'.
$query = DB::quote("'something's quoted'");

// 回傳 'something\'s quoted' 透過一個已定義的資料庫連線。
$query = DB::quote("'something's quoted'", $db_connection);

quote_identifier($string, $db)

quote_identifier 方法回傳一個為 SQL 查詢帶引號的字串。

靜態 是
參數
參數 預設 描述
$string 必要 要上引號的字串或字串陣列。
$db null 資料庫連線。
回傳 回傳一個為 SQL 查詢帶引號的字串。
範例 
// `users`.`name`
print_r(DB::quote_identifier('users.name'));

// array(
//    [0] => `users`.`name`
// )
print_r(DB::quote_identifier(array('users.name')));

list_columns($table, $like = null, $db = null)

list_columns 方法回傳一個在資料庫表所有行的清單。 選擇性的，一個 LIKE 字串可被用於搜尋指定的欄位。

靜態 是
參數
參數 預設 描述
$table 必要 要觀察的資料表。
$like null 要搜尋的行。
$db null 資料庫連線。
回傳 回傳一個帶有欄位資料的陣列。
範例 
/*
Array
(
	[id] => Array
	(
		[type] => int
		[min] => -2147483648
		[max] => 2147483647
		[column_name] => id
		[column_default] =>
		[data_type] => int
		[is_nullable] =>
		[ordinal_position] => 1
		[display] => 255
		[comment] =>
		[extra] => auto_increment
		[key] => PRI
		[privileges] => select,insert,update,references
	)

	[name] => Array
	(
		[type] => string
		[column_name] => name
		[column_default] =>
		[data_type] => varchar
		[is_nullable] =>
		[ordinal_position] => 3
		[character_maximum_length] => 255
		[collation_name] => utf8_unicode_ci
		[comment] =>
		[extra] =>
		[key] =>
		[privileges] => select,insert,update,references
	)
)
*/
print_r(DB::list_columns('users'));

/*
Array
(
	[name] => Array
	(
		[type] => string
		[column_name] => name
		[column_default] =>
		[data_type] => varchar
		[is_nullable] =>
		[ordinal_position] => 3
		[character_maximum_length] => 255
		[collation_name] => utf8_unicode_ci
		[comment] =>
		[extra] =>
		[key] =>
		[privileges] => select,insert,update,references

)

start_transaction($db = null)

start_transaction 方法在實例開始一個交易。

靜態 是
參數
參數 預設 描述
$db null 資料庫連線。
回傳 布林 - 只對 PDO 驅動程序有意義，所有其他的總是回傳 true， 但當 SQL 命令失敗時，會拋出一個 Exception。
範例

commit_transaction($db = null)

commit_transaction 方法送交所有等待中的交易查詢。

靜態 是
參數
參數 預設 描述
$db null 資料庫連線。
回傳 布林 - 只對 PDO 驅動程序有意義，所有其他的總是回傳 true， 但當 SQL 命令失敗時，會拋出一個 Exception。
範例 
DB::commit_transaction();

rollback_transaction($db = null, $rollback_all = true)

rollback_transaction 方法回滾所有等待中的交易查詢。

靜態 是
參數
參數 預設 描述
$db null 資料庫連線。
$rollback_all true 回滾模式：
true - 回滾並關閉交易機制；
false - 只在目前層級回滾。
回傳 布林 - 只對 PDO 驅動程序有意義，所有其他的總是回傳 true， 但當 SQL 命令失敗時，會拋出一個 Exception。
範例 
DB::rollback_transaction();

交易範例

// 典型的交易程式碼流程
try
{
    DB::start_transaction();

    // 一些查詢……

    DB::commit_transaction();

    // 回傳查詢結果
}
catch (Exception $e)
{
    // 回滾等待中的交易查詢
    DB::rollback_transaction();

    throw $e;
}
