View 類別

View 類別做為一個包裝嵌入式 PHP 的 HTML 頁面的物件，稱為「檢視」。 變數可以被指派給檢視物件，並且可以在檢視內本地引用。

閱讀更多關於檢視。

null
 檢視檔案名稱，如果沒提供你之後必須透過 set_filename 或在 render 中提供。檢視檔案應該被對應到 app/views 目錄。
$data 
null
 值的陣列

null
 設定為 true 或 false 來設定自動編碼，預設是主要配置設定（app/config/config.php）

$view = new View('path/to/view', array(
	'menu' => $menu,

forge($file = null, $data = null, $filter = null)

forge 方法回傳一個新的 View 物件。

靜態 是
參數
參數 預設 描述
$file 
null
 檢視檔案名稱，如果沒提供你之後必須透過 set_filename 或在 render 中提供。檢視檔案應該被對應到 app/views 目錄，並且不提供 .php 副檔名。
$data 
null
 值的陣列
$filter 
null
 設定為 true 或 false 來設定自動編碼，預設是主要配置設定（app/config/config.php）
回傳 一個新的 View 物件
範例 
// 將建立一個對 APPPATH/ 的 View 物件
$view = View::forge('path/to/view', array(
	'menu' => $menu,
	'articles' => $articles,
	'footer_links' => $footer_links,
));

auto_filter($filter = true)

auto_filter 方法設定在 View 實例上是否編碼資料。

靜態 否
參數
參數 預設 描述
$filter 
true
 在 View 實例上是否編碼資料
回傳 目前 View 物件
範例 
$view = View::forge('path/to/view', array(
	'menu' => $menu,
	'articles' => $articles,
	'footer_links' => $footer_links,
));

$view->auto_filter();

// 或設定它為 false

$view->auto_filter(false);

// 或鍊結

$view = View::forge('path/to/view', array(
	'menu' => $menu,
	'articles' => $articles,
	'footer_links' => $footer_links,
))->auto_filter();

set_filename($file)

set_filename 方法能讓你設定或變更檢視檔案名稱。

靜態 否
參數
參數 預設 描述
$file 必要 檢視檔案名稱，檢視檔案應該被對應到 app/views 目錄，並且不提供 .php 副檔名。
回傳 目前 View 物件
範例 
$view = new View();

$view->set_filename('path/to/view');

get($key = null, $default = null)

get 方法檢索一個或全部設定在檢視上的變數。

靜態 否
參數
參數 預設 描述
$key 
null
 變數名稱
$default 
null
 如果變數沒設定要回傳的預設值
回傳 混合
範例 
// 建立一個檢視並設定一些資料
$view = new View;
$view->set(array('field1' => 'value1', 'field2' => 'value2');

// 取得單一值
$field1 = $view->get('field1');

// 取得單一值，設定一個預設值如果該欄位不存在
$field3 = $view->get('field3', 'value3');

// 取得所有值
$fields = $view->get();

set($key, $value = null, $filter = null)

set 方法指派一個變數到檢視。

靜態 否
參數
參數 預設 描述
$key 必要 變數名稱或變數陣列
$value 
null
 值
$filter 
null
 設為 true 或 false 來編碼，預設是主要自動編碼設定
回傳 目前 View 物件
範例 
$view = new View;
$view->set('users', $users, false);

// 或設定一個陣列

$view->set(array(
	'users' => $users,
	'articles' => $articles,
), null, true);

// 當編碼輸出啟用時，
// 如果你需要傳遞 HTML／Javascript，你需要設定最後參數為 false。
$view->set('example', '<h1>Heading</h1>', false);

bind($key, $value = null)

透過參照指派一個值。綁定的好處是該值可以被修改，而不須重新設定它們。 也可能在變數有值之前綁定它們。在檢視檔案中， 指派值將可被做為一個變數。

請注意：這些不能被編碼輸出淨化，因此應該被認為是不安全的。

靜態 否
參數
參數 預設 描述
$key 必要 變數名稱或變數陣列
$value 
null
 值
回傳 目前 View 物件
範例 
$view = new View;
$view->bind('users', $users, false);

// 或設定一個陣列

$view->bind(array(
	'users' => $users,
	'articles' => $articles,
), null, true);

bind_global($key, $value = null)

bind_global 方法透過可存取的參照為所有檢視指派一個全域的變數。

請注意：這些不能被編碼輸出淨化，因此應該被認為是不安全的。

靜態 是
參數
參數 預設 描述
$key 必要 變數名稱
$value 必要 值
回傳 空
範例 
View::bind_global('users', $users);

render($file = null)

render 方法呈現檢視物件為一個字串。 全域和本地資料會合併並提取給檢視檔案，來建立本地變數。

靜態 否
參數
參數 類型 預設 描述
$file 字串 
null
 沒有副檔名的檢視檔案名稱
回傳 呈現的檢視檔案
範例 
$html = View::forge()->render('path/to/view');

// 或

$html = View::forge('path/to/view')->render();

// 或

$html = View::forge()->set_filename('path/to/view')->render();

程序助手

render($view, $data = array(), $auto_filter = null)

render 函式是 View::render 的別名。

參數
參數 類型 預設 描述
$view 字串 必要 沒有副檔名的檢視檔案名稱
$data 陣列 
array()
 要被傳遞給檢視的資料陣列
$auto_filter 布林 
null
 是否過濾資料
回傳 字串，來自 View::render 的結果
範例 
$html = render('path/to/view', $data_array);
