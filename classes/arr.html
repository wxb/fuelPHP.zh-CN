Arr 類別

Arr 類別是一系列與陣列協作的輔助函式。

is_multi($arr, $all_keys = false)

is_multi 方法檢查傳遞的陣列是否是多維陣列。

靜態 是
參數
參數 預設 描述
$arr 必要 要檢查的陣列。
$all_keys 
false
 檢查所有元素都是陣列。
回傳 布林
範例 
// 單一陣列
$arr = array('one' => 1, 'two' => 2);
echo Arr::is_multi($arr);
// 結果：false

// 多維陣列
$arr = array('one' => array('test' => 1), 'two' => array('test' => 2), 'three' => array('test' => 3));
echo Arr::is_multi($arr);
// 結果：true

// 多維陣列並檢查元素
$arr = array('one' => array('test' => 1), 'two' => array('test' => 2), 'three' => 3);

echo Arr::is_multi($arr, false); // 結果：true
echo Arr::is_multi($arr, true);  // 結果：false

is_assoc($arr)

is_assoc 方法檢查傳遞的陣列是否是關聯陣列。

靜態 是
參數
參數 預設 描述
$arr 必要 要檢查的陣列。
回傳 布林
範例 
$arr = array('foo', 'bar', 'baz', 'yay');
echo Arr::is_assoc($arr);
// 結果：false

$arr = array('foo' => 'foo', 'bar' => 'bar', 'baz' => 'baz', 'yay' => 'yay');
echo Arr::is_assoc($arr);
// 結果：true

/*
 * 請注意，即使 '2' 是定義做為一個字串，PHP 將在內部儲存它
 * 做為一個整數，因此這「不」被認為是一個關聯陣列！
 */
$arr = array(0 => 'foo', 1 => 'bar', '2' => 'baz', 3 => 'yay');
echo Arr::is_assoc($arr);
// 結果：false！

assoc_to_keyval($assoc, $key_field, $val_field)

assoc_to_keyval 方法轉換一個多維陣列成為一個鍵 => 值陣列。

靜態 是
參數
參數 預設 描述

	["Jill"] => 23

flatten($array, $glue = ':', $reset = true)

flatten 方法扁平化一個多維陣列（關聯和索引）成為一個一維陣列。

靜態 是
參數
參數 預設 描述
$array 必要 要扁平化的陣列。
$glue 
:
 要用來把鍵黏合在一起的字串
$reset 
true
 我們應該建立一個新值的陣列而非合併它們進入由 flatten 所建立最後的陣列？
回傳 陣列
範例

	array("a"),
	array("b"),
	array("c"),

// 結果：

(
	[0_0] => a
	[0_1] => b
	[0_2] => c
)

filter_keys($array, $keys, $remove = false)

filter_keys 方法過濾一個所給陣列成一組鍵。它回傳一個只包含在 $keys 陣列中項目的鍵的陣列。也可以從一個陣列移除指定的 $keys。

靜態 是
參數
參數 預設 描述
$array 必要 要過濾的陣列。
$keys 必要 要對以上陣列過濾的鍵陣列。
$remove 
false
 如果為 true，從 $array 移除 $keys 而非取回它們。
回傳 陣列
範例 
$arr = array(
	"user_name" => "John",
	"user_surname" => "Lastname",
	"project_name" => "Fuel",
	"project_type" => "Framework",
);

print_r(Arr::filter_keys($arr, array('project_name', 'user_name')));

// 結果：
Array
(
	[project_name] => Fuel
	[user_name] => John
)

// 讓我們移除一些鍵
print_r(Arr::filter_keys($arr, array('user_name', 'user_surname'), true));

// 結果：
Array
(
	[project_name] => Fuel
	[project_type] => Framework
)

filter_recursive($array, $callback = null)

filter_recursive 方法提供一個遞迴版本的 PHP array_filter() 函式。就像它的對應，你可以選擇性地傳遞一個回呼（callback）函式來決定什麼應該被過濾。

靜態 是
參數
參數 預設 描述
$array 必要 要過濾的陣列。
$callback 
null
 一個有效的回呼（callback）回傳一個布林。
回傳 陣列
範例 
$arr = array(
	"user_name" => "John",
	"user_surname" => "Lastname",
	"info" => array(
		0 => array(
			"data" = "a value",
		),
		1 => array(
			"data" = "",
		),
		2 => array(
			"data" = 0,
		),
	),
);

print_r(Arr::filter_recursive($arr));

// 結果：
Array
(
	[project_name] => Fuel
	[user_name] => John
	[info] => Array
	(
		[0] => Array
		(
			[data] => a value
		)
	)
)

// 讓我們使用一個回呼（callback）
print_r(Arr::filter_recursive($arr, function($item){ return $item !== ""; });

// 結果：
Array
(
	[project_name] => Fuel
	[user_name] => John
	[info] => Array
	(
		[0] => Array
		(
			[data] => a value
		)
		[2] => Array
		(
			[data] => 0
		)
	)
)

subset($array, $keys, $default = null)

與 filter_keys 相似，subset 方法使用 $keys 中的值來存取此陣列並回傳關聯的值。不同的地方是 subset 回傳所有請求的鍵，為任何缺少的鍵提供 $default。

這個方法內部使用 get 和 set，所以 $keys 可以包含句點表示法。

靜態 是
參數
參數 預設 描述
$array 必要 要篩選的陣列。
$keys 必要 要篩選上面陣列的鍵的陣列。
$default 
null
 在 $array 中找不到的鍵會給這個值來代替。
回傳 陣列
範例 
$arr = array(
	"user" => array(
		"name" => "John",
		"surname" => "Lastname",
	),
	"project" => array(
		"name" => "Fuel",
		"type" => "Framework",
	),
);

print_r(Arr::subset($arr, array('project.name', 'user.name')));

// 結果：
Array
(
 	[project] => Array
         (
			 [name] => Fuel
		 )
	[user] => Array
         (
			 [name] => John
		 )
)

print_r(Arr::subset($arr, array('project.name', 'project.manager')));

// 結果：
Array
(
 	[project] => Array
         (
			 [name] => Fuel
			 [manager] =>
		 )
)

print_r(Arr::subset($arr, array('project.name', 'project.manager', 'user', 'not_provided'), 'Not Provided'));

// 結果：
Array
(
 	[project] => Array
         (
			 [name] => Fuel
			 [manager] => Not Provided
		 )
 	[user] => Array
         (
			 [name] => John
			 [surname] => Lastname
		 )
	[not_provided] => Not Provided
)

get($array, $key, $default = false)

get 方法回傳所給使用句點表示法陣列的一個元素，或預設值如果它沒被設定的話。

靜態 是
參數
參數 預設 描述
$array 必要 要存取的陣列
$key 必要 在 $array 中請求的鍵。如果 null 被傳遞做為鍵，整個陣列會被回傳。
$default 
false
 如果請求的鍵不存在時要被回傳的值
回傳 混合。如果你傳遞一個鍵的陣列，回傳值將會是一個有所有請求刪除結果的陣列。
範例 
$person = array(
	"name" => "Jack",
	"age" => "21",
	"location" => array(
		"city" => "Pittsburgh",
		"state" => "PA",
		"country" => "US"
	)
);

echo Arr::get($person, "name", "Unknown Name");
// 結果："Jack"

echo Arr::get($person, "job", "Unknown job");
// 結果："Unknown job"

// 這個方法也可以透過使用一個點來分隔鍵與鍵之間，來深入到陣列
echo Arr::get($person, "location.city", "Unknown City");
// 結果："Pittsburgh"

set(&$array, $key, $value = null)

set 方法設定所給使用句點表示法陣列的元素。

警告：原始陣列透過參照被編輯

靜態 是
參數
參數 預設 描述
$array 必要 要存取的陣列
$key 必要 在 $array 中請求的鍵。如果沒有傳遞鍵，或 null 被傳遞做為鍵，該陣列會取消設定。如果你傳遞一個鍵 - 值配對的陣列，所有陣列中的值將被設定。
$value 
null
 要被設定的值。如果 $key 是一個陣列會被忽略。
回傳 空
範例 
$person = array(
	"name" => "Jack",
	"age" => "21",
	"location" => array(
		"city" => "Pittsburgh",
		"state" => "PA",
		"country" => "US"
	)
);

Arr::set($person, "name", "John");
// $person['name'] 設為 "John"

// 這個方法也可以透過使用一個點來分隔鍵與鍵之間，來深入到陣列
Arr::set($person, "location.city", "Philadelphia");
// $person['location']['city'] 設為 "Philadelphia"

// 或一次設定多個值
Arr::set($person, array("name" => "John", "location.city" => "Philadelphia"));

pluck($array, $key, $index = null)

pluck 方法從一個陣列的集合或物件採集值。

靜態 是
參數
參數 預設 描述
$array 必要 要採集的來源陣列
$key 必要 要採集來源陣列的鍵。
$index 
null
 選擇性的回傳陣列索引。
回傳 空
範例 
$collection = array(
	array(
		'id' => 2,
		'name' => 'Bill',
		'surname' => 'Cosby',
	),
	array(
		'id' => 5,

		'surname' => 'Rock',
	),
	array(
		'id' => 7,
		'name' => 'Bert',
		'surname' => 'Visser',
	),
);

// 取得一個 ID 的陣列
$ids = \Arr::pluck($collection, 'id');

// 取得一個名稱以 ID 為索引的陣列
$names = \Arr::pluck($collection, 'name', 'id');
// array(2 => 'Bill', 5 => 'Chris', 7 => 'Bert');

delete(&$array, $key)

delete 方法刪除所給使用句點表示法陣列的元素。
警告：原始陣列透過參照被編輯，只有回傳成功的布林值

靜態 是
參數
參數 預設 描述
$array 必要 要存取的陣列
$key 必要 請求在 $array 中的鍵。如果沒有傳遞鍵，或傳遞 null 做為鍵值，該陣列會取消設定。
回傳 混合，true 如果該鍵被刪除，false 如果該鍵不存在。如果你傳遞一個鍵的陣列，回傳值將會是一個有所有請求刪除結果的陣列。
範例 
$person = array(
	"name" => "Jack",
	"age" => "21",
	"location" => array(
		"city" => "Pittsburgh",
		"state" => "PA",
		"country" => "US"
	)
);

$result = Arr::delete($person, "name");
// 從該陣列刪除 $person['name']，回傳 true

// 這個方法也可以透過使用一個點來分隔鍵與鍵之間，來深入到陣列
$result = Arr::delete($person, "location.city");
// 從該陣列刪除 $person['location']['city']，回傳 true

// 或一次刪除多個值
$result = Arr::delete($person, array("name", "location.doesnotexist"));
// 刪除 $person['name']，回傳 array('name' => true, 'location.doesnotexist' => false)

insert 方法主要是一個 array_splice 的別名，添加了錯誤檢查
警告：原始陣列透過參照被編輯，只有回傳成功的布林值

靜態 是
參數
參數 預設 描述
$original 必要 要使用的陣列
$value 必要 要新增的（多個）值
$pos 必要 要插入的數字位置，負數會從結尾向後算
回傳 布林
範例

print_r($people);
// 結果：
Array
(
	[0] => Humpty
	[1] => Jack
	[2] => Jill
)

// 添加多個值
Arr::insert($people, array("Hansel", "Gretel"), 1);
print_r($people);
// 結果：
Array
(
	[0] => Humpty
	[1] => Hansel
	[2] => Gretel
	[3] => Jack
	[4] => Jill
)

// 添加一個陣列
Arr::insert($people, array( array("name" => "Wolf", "teeth" => "sharp")), 0);
print_r($people);

// 結果：
Array
(
