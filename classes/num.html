Num 類別

Num 類別提供額外的格式化方法與數值協作

配置

類別提供不同的方法以格式化某些類型的數字。 格式是字串，其中 0 代表回傳值的數字部分。

請注意：當變更 Num 配置，複製檔案至 app/config/num.php 並變更設定，這能易於升級而不失去你的自訂設定。

鍵 類型 預設 描述
phone 字串 
'(000) 000-0000'
 預設的電話號碼格式。
smart_phone 陣列 
array(
	7  => '000-0000',
	10 => '(000) 000-0000',
	11 => '0 (000) 000-0000',
),
 基於號碼長度的電話號碼格式。（長度 => 格式）
exp 字串 
'00-00'
 信用卡到期日的預設格式。
credit_card 字串 
'00-00'
 信用卡號碼遮罩預設格式。

bytes($size = 0)

bytes 方法轉換數字（檔案大小）為位元組值。 檔案大小定義的格式為：SB，其中 S 是大小（1、8.5、300等），B 是位元組單位（K、MiB、GB、等）。 所有有效的位元組單位被定義於 core/lang/en/byte_units.php

靜態 是
參數
參數 類型 預設 描述
$size 字串｜整數｜浮點數 
0
 要轉換的字串。
回傳 浮點數
範例 
echo Num::bytes('200K');  // 204800
echo Num::bytes('5MiB');  // 5242880
echo Num::bytes('1000');  // 1000
echo Num::bytes('2.5GB'); // 2684354560

format_bytes($bytes = 0, $decimals = 0)

format_bytes 方法轉換位元組數字為可讀的格式。

靜態 是
參數
參數 類型 預設 描述
$bytes 整數 
0
 要格式化的數字。
$decimals 整數 
0
 要四捨五入的小數位數。
回傳 字串｜false
範例 
echo Num::format_bytes('204800');     // 200 kB
echo Num::format_bytes('214901', 1);  // 209.9 kB
echo Num::format_bytes('2249010', 1); // 2.1 MB
echo Num::format_bytes('badbytes');   // false

$decimals 整數

format($string = '', $format = '')

format 方法轉換數字為所給的格式。

靜態 是
參數
參數 類型 預設 描述
$string 字串 
''
 要格式化的數字。
$format 字串 
''
 格式。
回傳 字串
範例 
echo Num::format('1234567890', '(000) 000-0000'); // (123) 456-7890
echo Num::format('1234567890', '000.000.0000');   // 123.456.7890
echo Num::format('1234567890', '000.000.0000');   // 123.456.7890
echo Num::format('1', '000.000.000');             // 1..

mask_string 方法轉換數字為所給的格式。

靜態 是
參數

$string 字串

$format 字串 
''

$ignore 字串

format_phone($string = '', $format = null)

format_phone 方法轉換數字為所給的格式。

靜態 是
參數
參數 類型 預設 描述
$string 字串 
''
 要格式化的數字。
$format 字串 
null
 格式。當 null 時預設指向 format_phone 配置設定。
回傳 字串
範例

smart_format_phone($string)

smart_format_phone 方法基於號碼長度， 轉換數字為電話號碼格式。

請注意： 數字依據長度被格式化， 這些定義在 num 配置。

靜態 是
參數
參數 類型 預設 描述
$string 字串 必要 要格式化的數字。
回傳 字串
範例 
echo Num::smart_format_phone('1234567');
// 123-4567

echo Num::smart_format_phone('01234567890');
// 0 (123) 456-7890

format_exp 方法轉換數字為信用卡到期日字串。

靜態 是
參數

$format 字串 
null
 格式。當 null 時預設指向 exp 配置設定。
