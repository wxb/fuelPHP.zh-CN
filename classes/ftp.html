Ftp 類別

FTP 類別能讓你透過 FTP 協定對遠端伺服器上傳、下載、移動和鏡像檔案。

配置

FTP 類別透過 fuel/core/config/ftp.php 配置檔案進行配置。它已經填充了一個預設配置群組。你可以藉由複製此配置檔案到你的應用程式 config 目錄，並根據需求修改該檔案，以覆寫此配置群組並添加新群組。

以下配置設定可以被定義：

參數 類型 預設 描述
hostname 字串 
'localhost'
 要連接的 FTP 伺服器的 IP 或網域名稱。
username 字串 
''
 選擇性：要連接的使用者名稱，如果登入是必要的話。
password 字串 
''
 選擇性：要連接的使用者密碼，如果登入是必要的話。
timeout 整數 
90
 所有後續的網路操作超時（秒）。
port 整數 
21
 你的 FTP 伺服器回應的連接埠號。大多數伺服器使用 21。
passive 布林 
true
 是否使用被動模式。預設情況下被動模式自動被設定。
ssl_mode 布林 
false
 使用比通常的 FTP 稍微安全一點的 FTPS。（注：這不是 SFTP）。
debug 布林 
false
 是否啟用除錯以顯示錯誤訊息。

forge($config = 'default', $connect = true)

forge 方法被用於建立一個新的 FTP 類別實例，可以是不同配置群組的參照， 或傳遞一個 配置選項 的陣列。

靜態 是
參數
參數 預設 描述
$config 
'default'
 要使用的配置群組名稱，或一個配置陣列。
$connect 
true
 自動連接到此伺服器。
回傳 Ftp 物件
範例 
// 連接到預設伺服器
$ftp = Ftp::forge();

// 連接到一個預定義在 config/ftp.php 的伺服器
$ftp2 = Ftp::forge('image-storage');

// 動態連接到一個伺服器
$ftp3 = Ftp::forge(array(
	'hostname' => 'fuelphp.com',
	'username' => '',
	'password' => '',
	'timeout'  => 90,
	'port'     => 21,
	'passive'  => true,
	'ssl_mode' => false,
	'debug'    => false
));

if ($ftp3->delete_dir('/') === true)
{
	exit('The world owes you a debt of gratitude');
}
else
{
	exit('You tried and that is the main thing.');
}

connect()

connect 方法能讓你手動連接到一個 FTP 資源。 你只在當你 forge() ftp 物件而不自動連線時使用此方法。

靜態 否
參數 無
回傳 目前 FTP 物件，鍊結用
範例 
// 建立一個 ftp 物件，但不連線
$ftp = Ftp::forge(array(
	'hostname' => 'ftp.example.com',
	'username' => '',
	'password' => '',
	'timeout'  => 90,
	'port'     => 21,
	'passive'  => true,
	'ssl_mode' => false,
	'debug'    => false
), false);

// 在這裡做一些東西

// 現在連接到伺服器
$ftp->connect();

change_dir($path = '')

change_dir 變更「目前目錄」。

靜態 否
參數
參數 預設 描述
$path 
''
 要移往的遠端路徑。
回傳 布林
範例 
// 移往 /user/phil/public_html/some/path/
$ftp->change_dir('/public_html/some/path/');

mkdir($path, $permissions = null)

mkdir 方法用於在遠端伺服器建立一個新目錄。

靜態 否
參數
參數 預設 描述
$path 必要 要建立的遠端目錄。
$permissions 
null
 如果設定，權限將被應用到該目錄。
回傳 布林
範例 
// 建立一個可寫入的上傳文件夾
$ftp->mkdir('/public_html/uploads/', 0777);

upload($local_path, $remote_path, $mode = 'auto', $permissions = null)

從本地伺服器上傳一個檔案或目錄到遠端伺服器。

靜態 否
參數
參數 預設 描述
$local_path 必要 本地伺服器路徑。
$remote_path 必要 遠端伺服器路徑。
$mode 
'auto'
 選項有 ascii、binary、和 auto（預設）。 如果使用 auto，它將依據來源檔案的副檔名使用模式。
$permissions 
null
 如果設定，權限將被應用到該目錄。
回傳 布林
範例 
// 上傳 myfile.html
$ftp->upload('/local/path/to/myfile.html', '/public_html/myfile.html', 'ascii', 0775);

download($remote_path, $local_path, $mode = 'auto')

從遠端伺服器下載一個檔案或目錄到本地伺服器。

靜態 否
參數
參數 預設 描述
$remote_path 必要 遠端伺服器路徑。
$local_path 必要 本地伺服器路徑。
$mode 
'auto'
 選項有 ascii、binary、和 auto（預設）。 如果使用 auto，它將依據來源檔案的副檔名使用模式。
回傳 布林
範例 
// 下載 myotherfile.html
$ftp->download('/public_html/myfile.html', '/local/path/to/myfile.html', 'ascii');

rename($old_file, $new_file, $move = false)

在遠端伺服器移動或更名一個檔案。

靜態 否
參數
參數 預設 描述
$old_file 必要 要被更名或移動的檔案。
$new_file 必要 移動或更名為目標的檔案。
$move 
false
 'true' 如果這是移動，'false' 如果是一個更名。 此標誌只用於錯誤訊息除錯。
回傳 布林
範例 
// 連線到一個 FTP 伺服器
$ftp = Ftp::forge($config);

// 更名檔案
$ftp->rename('/path/to/oldfile.txt', '/path/to/newfile.txt', false);

// 關閉連線
$ftp->close();

move($old_file, $new_file)

rename() 的別名，帶著 $move 標誌設為 true。

delete_file($filepath)

從遠端伺服器刪除一個檔案。

靜態 否
參數
參數 預設 描述
$filepath 必要 要被刪除的檔案。
回傳 布林
範例 
// 連線到一個 FTP 伺服器
$ftp = Ftp::forge($config);

// 刪除一個檔案。
if ( ! $ftp->delete_file('/path/to/file.txt'))
{
	// 刪除失敗
}

// 關閉連線
$ftp->close();

delete_dir($filepath)

從遠端伺服器刪除一個目錄並遞迴在其中的任何東西。

靜態 否
參數
參數 預設 描述
$filepath 必要 要被刪除的檔案。
回傳 布林
範例 
// 連線到一個 FTP 伺服器
$ftp = Ftp::forge($config);

// 刪除一個目錄及所有它的內容
if ( ! $ftp->delete_dir('/path/to/folder'))
{
	// 刪除失敗
}

// 關閉連線
$ftp->close();

chmod($path, $perm)

在遠端伺服器變更一個檔案的權限。

靜態 否
參數
參數 預設 描述
$path 必要 要變更權限的檔案。
$perm 必要 權限，unix 風格定義，為一個八進位值。
回傳 布林
範例 
// 連線到一個 FTP 伺服器
$ftp = Ftp::forge($config);

// 使檔案為全部可寫
if ( ! $ftp->chmod('/path/to/file.txt', 0666))
{
	// 變更權限失敗
}

// 關閉連線
$ftp->close();

list_files($path = '.')

表列在遠端伺服器上所給路徑中的所有檔案。

靜態 否
參數
參數 預設 描述
$path 
'.'
 遠端伺服器路徑。預設為目前目錄。
回傳 陣列
範例 
// 連線到一個 FTP 伺服器
$ftp = Ftp::forge($config);

// 表列檔案
if ( $files = $ftp->list_files('/path/to/files') === false)
{
	// 表列失敗
}
else
{
	/* 可能回傳一些東西像
	   $files = array(3) {
		  [0]=>
		  string(11) "public_html"
		  [1]=>
		  string(10) "public_ftp"
		  [2]=>
		  string(3) "www"
	*/
  }

// 關閉連線
$ftp->close();

mirror($local_path, $remote_path)

遞迴讀取本地文件夾與其包含的一切（包括子文件夾），並藉由 FTP 的基礎上建立一個鏡像。 無論原始檔案路徑的目錄結構，將在伺服器上被重建。

靜態 否
參數
參數 預設 描述
$local_path 
'default'
 本地伺服器路徑。
$remote_path 
true
 遠端伺服器路徑。
回傳 布林
範例 
$ftp = Ftp::forge();

$ftp->connect($config);

$ftp->mirror('/path/to/myfolder/', '/public_html/myfolder/');

$ftp->close();

close()

close 方法關閉一個 forge() 或 connect() 建立的開啟到遠端伺服器的 ftp 連線。

靜態 否
參數 無
回傳 布林
範例 
// 關閉一條連線
if ( ! $ftp->close())
{
	// 連線未開啟
}
