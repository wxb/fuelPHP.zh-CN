Fuel 類別

Fuel 類別包含 Fuel 框架的核心方法。 此類別能使用在你的應用程式的公開方法都記載在此。 所有其他的方法僅供內部使用，且不應被使用。

類別常數

環境常數，這些被用於你的 config.php 以定義應用程式環境：

名稱 類型 值 描述
TEST 字串 
'test'
 識別 TEST 環境。
DEVELOPMENT 字串 
'development'
 識別 DEVELOPMENT 環境。
STAGING 字串 
'staging'
 識別 STAGING 環境。
PRODUCTION 字串 
'production'
 識別 PRODUCTION 環境。

load($file)

load 方法載入（引入）一個所給的檔案，並回傳結果。例如這個方法被用於載入配置檔案，會回傳一個配置設定的陣列。

靜態 是
參數
參數 預設 描述
$file 必要 要被載入的檔案名稱
回傳 混合
範例 
// 載入一個 PHP 檔案
if (Fuel::load('myfile.php'))
{
	echo "MyFile was loaded successfully";
}

請注意，如果引入的文件不包含一個 return 語句，load() 如果載入成功將回傳 1，或是檔案無法被載入回傳 0！
